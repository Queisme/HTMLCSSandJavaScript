HTML provides four phrasing elements to emphasize text within the body of a document

  1. Text enclosed between -b- tags are enhanced without conveying extra importance - just visual
  2. Text enclosed between -i- tags is enhanced without conveying extra importance - just visual
  3. Text enclosed between -strong- tags gains increased importance
  4. Text enclosed between -em- tags gains increased importance

Use -strong- and -em- tags. The -b- tag should be avoided wherever possible, but one legitimate use is to mark up the lead sentence of an article.

Warning: This dungeon is dangerous. Avoid the ducks. Take any gold you find. Do not take any diamonds, they are explosive. You have been warned.

Puppy dogs are cute.

Puppy dongs are cute.

Puppy dogs are cute.
